# 🧶 Tati Encantos

Bem-vindo(a) ao repositório do projeto **Tati Encantos**! 

Este projeto é uma vitrine digital completa desenvolvida para destacar o trabalho artesanal, oferecendo uma experiência de navegação agradável e intuitiva para os clientes conhecerem e adquirirem os produtos.

## 💻 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout responsivo e animações para uma interface atraente.
* **JavaScript:** Interatividade da página e dinamismo da interface.

## ✨ Funcionalidades

* Catálogo de produtos organizado de forma visual.
* Design responsivo (adapta-se perfeitamente a celulares, tablets e computadores).
* Interface amigável e focada na experiência do usuário (UX).
